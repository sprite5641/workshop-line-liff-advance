# Install the Netlify CLI
````
$ npm install -g netlify-cli
````

# Create a new site in Netlify
````
$ ntl init
````

# Deploy to a unique preview URL
````
$ npm run build
````
````
$ ntl deploy
````
# Deploy the site into production
````
$ ntl deploy --prod
````
